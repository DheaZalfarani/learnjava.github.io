Studi Kasus : Pengkondisian Boolean

Pada praktik ini kita akan mencoba bagaimana menggunakan blok pengkondisian. Ada beberapa variabel yang akan digunakan untuk mendemonstrasikan praktik ini. Silahkan susun kode logika AND dengan variabel yang akan menampilkan hasil benar atau salah dan gunakan blok if else seperti gambar berikut.

Setelah selesai mengerjakan, klik jalankan terlebih dahulu. Struktur kode dan output akan muncul, kemudian silahkan mengecek jawaban pada tombol cek jawaban.
